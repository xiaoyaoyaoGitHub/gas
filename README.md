

# 当前main分支  
# 用于开发云信的微信小程序、支付宝小程序。云信小程序截止2024年11月12日，功能有：加油、车检


### 技术栈
vue2 + uview2 UI

### 项目介绍
1.基于uniapp，兼容微信、抖音、支付宝小程序，为保证兼容性，故选择上述技术栈
2.支付宝小程序和微信小程序主体都是是云信

### 代码注意事项
1.uView 的时间选择器，修改了源代码的样式，需注意
2.这个   CarTypeSelector.vue   组件，是用于车检下单界面，选择车辆类型时使用的。最初的需求，车辆类型有两层要选择，后边改需求为只选一层。为避免后期又改回来，故没有修改代码，而是沿用了组件代码。
3.当前首页，“车检列表页”是首页，第一次首页进入时，在onShow中就请求了一次列表数据，用于用户在每次切换页面时，都是获取的最新数据。随后onLoad中内获取用户定位后，再次获取列表数据。
 （当onLoad和onShow都是异步函数时，此时的执行顺序是看谁先请求完）
